How To Run :-

1. Change MQTT_SERVER variable to the ip of network used 

2. Run suscriber.py
	python suscriber.py

3. Connect Pi with Aurdino

4. Run publish.py to start publishing information on server
	python publish.py

5. Open MY MQTT APP on Playstore
6. Go to Settings and set Broker Url to the ip of the network used
7. Suscribe to channels of your choice from the list below
   .) notification 
   .) sensor_value
   .) plant_health
   .) debug
